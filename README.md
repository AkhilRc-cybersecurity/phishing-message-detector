# Phishing Message Detector

## Description
A Python-based tool that detects phishing messages using keyword-based analysis.

## Features
- Detects suspicious phishing messages
- Simple and beginner-friendly logic
- Command-line based

## Technologies Used
- Python

## How to Run
1. Download the repository
2. Run `python phishing_detector.py`
3. Enter a message to check
